# Baton Stack Configs

`reeve-egress.yaml` is generated by `cd ~/Code/reeve && make ledger-publish`.
Do not edit it by hand; update Reeve's `ledger-schemas/` and publish again.

The root `baton.yaml` references `reeve-smoke.yaml` from node metadata
until Baton grows first-class multi-health-check polling. Current Baton
parses metadata as opaque strings; consumers that understand this config
must load the referenced file explicitly.
